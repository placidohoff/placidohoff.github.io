Fight Scorecards Application

A useful web application for fight fans such as myself. This app allows fans to score a fight and to save notes for each round if they choose. This app also connects to a database which allows the user to save the previous scorecards and notes that they might want to view at a future time. This app was built with the React.js javascript framework.

Note Taking Application

Full stack application that allows the user to log in and create for example a chapter by chapter summary of a book or subject of their choice. The notes can be saved and resumed whenever the user decides to log in again. Each user can hold as many notes in their collection to be viewed and edited as they please. Built with React.js front-end and firebase database backend.
